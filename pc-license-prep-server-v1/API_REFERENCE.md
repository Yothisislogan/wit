# API Reference V1

- GET / serves the frontend
- GET /api/health returns status
- GET /api/modules returns course modules
- GET /api/modules/{module_id} returns one module
- GET /api/questions returns practice questions
- POST /api/questions/submit scores answers
- GET /api/flashcards returns terms
- GET /api/progress returns local progress summary
- POST /api/progress saves local progress
